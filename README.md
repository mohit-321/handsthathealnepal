# Hands That Heal Nepal 🇳🇵
A crypto donation website for elderly &amp; disabled communities in rural Morang, Nepal. Anonymous campaign. Accepts BTC, ETH, SOL &amp; USDT.
A humanitarian crypto donation campaign supporting elderly and disabled 
individuals in rural Morang District, Nepal.
## About
- 100% of donations go directly to those in need
- Accepts Bitcoin, Ethereum, Solana & USDT (TRC20)
- Anonymous campaign — no organisation, no overhead
- Built with plain HTML/CSS/JS — no frameworks, no tracking
